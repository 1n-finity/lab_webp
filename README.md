## üöÄ Getting Started

### Viewing the Website

1. **Local File Access**: 
   - Navigate to the website folder
   - Open `index.html` in your web browser
   - Navigate between pages using the menu

2. **Local Server** (Recommended):
   ```bash
   # If you have Python installed
   python -m http.server 8000
   
   # Or if you have Node.js
   npx serve .
   
   # Then visit http://localhost:8000
   ```

### Customization Guide

#### Adding New Pages
1. Create new HTML file following the existing template structure
2. Copy header, navigation, and footer from existing pages
3. Add the new page link to navigation in all HTML files
4. Update CSS if needed for page-specific styles

#### Updating Content
- **Profile Information**: Edit hero section in `index.html` and about section in `about.html`
- **Research Areas**: Modify research cards and projects in `research.html`
- **Experience**: Update timeline items in `experience.html`
- **Contact Details**: Change contact information in `contact.html`

#### Styling Changes
- **Colors**: Update CSS custom properties in `styles.css`
- **Typography**: Change font imports and font-family declarations
- **Layout**: Modify grid and flexbox properties for different layouts
