Lab Resources

Information about our computational infrastructure, software tools, and shared resources available to group members and collaborators.

Computational Clusters

  • High-Performance Compute Node A

    20-core Intel Xeon CPU, 128GB RAM, NVIDIA RTX 5090 GPU. Used for routine DFT calculations.

  • High-Performance Compute Node B

    8-core Intel Xeon CPU, 32GB RAM, NVIDIA RTX A4000 GPU.

  • Workstations

    Dedicated high-end workstations for data analysis, visualization, and test runs.

Software & Tools

  • VASP / Gaussian

    Licenses and access to major quantum chemistry and solid-state physics packages.

  • Materials Studio

    Tools for creating molecular/crystal models and analyzing computational outputs.

  • In-house Scripts

    Custom Python/Bash scripts for automating workflows and high-throughput screening data extraction.
